# Project Overview
Analysis of Germany's electricity consumption and renewable energy generation (2015-2020) using SQL and Python. Data source: Open Power System Data.

# Key Findings

Wind energy generation grew by 61% from 8,809 MW (2015) to 14,178 MW (2020)
Solar generation increased from 3,986 MW to 6,304 MW over the same period
Electricity consumption dropped in 2020 due to COVID-19 (-5.2% vs 2019)
February has the highest average consumption (59,974 MW), May the lowest (52,285 MW)
Forecast error spiked in 2016 and 2019, while 2020 showed negative error due to reduced demand

# Tech Stack

Python, SQLite, Pandas, Matplotlib, DBeaver

# Data Source

Open Power System Data — Time series 60min
